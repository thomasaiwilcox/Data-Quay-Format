# COVE-CX Bitstream Vectors

This directory contains stable companion-vector descriptors for COVE-owned registered codecs:

- `org.coveformat.codec.fsst-utf8.v2`
- `org.coveformat.codec.alp-float.v2`
- `org.coveformat.codec.fastlanes-integer.v2`

The reference implementation exposes the stable descriptors and validates registered-payload/fallback logical equivalence through `cove-codec`.
